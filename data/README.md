# Data

Source: IBM sample dataset via Kaggle
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Download `WA_Fn-UseC_-Telco-Customer-Churn.csv` into this folder before
running the pipeline. 7,043 rows, 21 columns, one row per customer.
The raw file is gitignored to keep the repo lean; this README documents
provenance instead.
